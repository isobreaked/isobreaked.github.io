Problem 1746 - 2022 AMO, P4 of 8

Difficulty: 6

Let \(S\) be the set of points \((i,j)\) in the plane with \(i,j \in \{1,2,\dots,2022\}\) and \(i < j\). We would like to colour each point in \(S\) either red or blue, so that whenever \((i,j)\) and \((j,k)\) have the same colour, the point \((i,k)\) also has that colour. How many such colourings of \(S\) are there?
