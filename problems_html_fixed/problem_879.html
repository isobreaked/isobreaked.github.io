Problem 879 - 2020 IMOSL, N5, N

Difficulty: 8

Determine all functions \(f\) defined on the set of all positive integers and taking non-negative integer values, satisfying the three conditions:

  1. \(f(n) \neq 0\) for at least one \(n\);
  2. \(f(x y)=f(x)+f(y)\) for every positive integers \(x\) and \(y\);
  3. there are infinitely many positive integers \(n\) such that \(f(k)=f(n-k)\) for all \(k < n\).
