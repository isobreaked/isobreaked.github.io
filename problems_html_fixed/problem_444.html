Problem 444 - 1994 BMO2, P4 of 4, N

Difficulty: 6

How many perfect squares are there mod \(2^n\)?

Thinking

Suppose \(x^2 \equiv y^2 \pmod{2^n}\) for \(x \not \equiv y \pmod{2^n}\). Then \(2^n \mid (x + y)(x - y)\). Suppose \(x\) and \(y\) are both odd. Then \((x + y) + (x - y) = 2x\) which means that at least one of \(x + y\) and \(x - y\) has a \(\nu_2\) of \(1\). Thus the other one must have a \(\nu_2\) of \(n-1\), or \(n\). This means that \(y \equiv x + 2^{n-1}\) must hold, or \(y \equiv -x\) so there are \(2^{n-3}\) squares originating from the odd moduli (apart from when \(n \in \{2, 1, 0\}\)).

A clever trick to figure out the number of even squares here is to note that there is a clear recursion - the number of even squares for \(2^n\) is the number of total squares for \(2^{n-2}\). Thus \[T_{2N} = O_{2N} + T_{2N - 2} = O_{2N} + O_{2N-2} + T_{2N-4} = O_{2N} + O_{2N - 2} + O_{2N-4} + \dots + O_4 + T_2 = 2 + \sum_{i=2}^{N} O_{2i} = 2 + \sum_{i=2}^N 2^{2i-3} = \frac{2^{2N-1} + 4}{3}\] \[T_{2N+1} = O_{2N+1} + O_{2N-1} + \dots + O_5 + T_3 = 3 + \sum_{i=2}^{N} O_{2i+1} = 3 + \sum_{i=2}^N 2^{2i-2} = \frac{2^{2N} + 5}{3}\]

Idea

Convert to simpler problems! If you see something that looks similar to something else (maybe a subproblem), it's usually the right intuition - isomorphism is real.
