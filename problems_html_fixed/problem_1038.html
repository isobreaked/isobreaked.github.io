Problem 1038 - 2002 USAMO, P4 of 6

Difficulty: 6

Let \(\mathbf{R}\) be the set of real numbers. Determine all functions \(f : \mathbf{R} \rightarrow \mathbf{R}\) such that \[f(x^2 - y^2) = xf(x) - yf(y)\] for all pairs of real numbers \(x\) and \(y\).

Thinking

We have \(f(x^2) = xf(x)\), so \(f(x^2 - y^2) = f(x^2) - f(y^2)\). Also, \(f(-x^2) = -xf(x)\), so \(f(-x) = -f(x)\).

So we have cauchy (probably) and \(f(x^2) = xf(x)\) \[xf(x)+yf(y)+xf(y)+yf(x)= (x + y)f(x+y) = f((x+y)^2) = f(x^2) + f(y^2) + f(2xy) = xf(x) + yf(y) + 2f(xy)\] Letting \(g(x) = f(x)/x\), we get \[g(x) + g(y) = 2g(xy) \implies g(1) = g(x) \;\; \forall x \implies f(x) = cx \;\; \forall x\]

Idea

Cauchy with most general things is going to be very powerful. Cauchy is treated here as a `plug-in', adding another variable to the \(f(x^2) = xf(x)\) equation to equate in two ways.
