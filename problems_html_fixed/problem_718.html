Problem 718 - 2015 IMOSL, N6, N

Difficulty: 8

Let \(\mathbf{Z}_{>0}\) denote the set of positive integers. Consider a function \(f: \mathbf{Z}_{>0} \to \mathbf{Z}_{>0}\). For any \(m, n \in \mathbf{Z}_{>0}\) we write \(f^n(m) = \underbrace{f(f(\ldots f}_{n}(m)\ldots))\). Suppose that \(f\) has the following two properties:

  • if \(m, n \in \mathbf{Z}_{>0}\), then \(\frac{f^n(m) - m}{n} \in \mathbf{Z}_{>0}\);
  • The set \(\mathbf{Z}_{>0} \setminus \{f(n) \mid n\in \mathbf{Z}_{>0}\}\) is finite.
Prove that the sequence \(f(1) - 1, f(2) - 2, f(3) - 3, \ldots\) is periodic.
