Problem 1496 - 2002 IMOSL, A2

Difficulty: 7

Let \(a_1,a_2,\ldots\) be an infinite sequence of real numbers, for which there exists a real number \(c\) with \(0\leq a_i\leq c\) for all \(i\), such that\[\left\lvert a_i-a_j \right\rvert\geq \frac{1}{i+j} \quad \text{for all }i,\ j \text{ with } i \neq j.\]Prove that \(c\geq1\).

Thinking

Let \(f : \mathbf{N} \to \mathbf{R}\) such that \(f(n) = a_n\) for all \(n\). Then we know \(f\) is bounded above, with \(0 \le f(i)\le c\) for all \(i\) and for all \(i \neq j\), \[| f(i) - f(j) | \ge \frac{1}{i+j}\] This is very weird. It doesn't make intuitive sense for now. Okay let's look at \(f(n)\), \(f(n+1)\) and \(f(n+2)\). Basically, if \(f(n)\) to \(f(n+1)\) is in one direction and \(f(n+1)\) to \(f(n+2)\) is in the other direction, then the \(f(n)\) to \(f(n+2)\) bound is somewhat stronger than the other two combined in some cases.

Oh, but of course that is true - the bound if \(f(k)\) is in the middle of \(f(i)\) and \(f(j)\) is obviously (? might be untrue) better if we use \(f(k)\). Concretely: \[\frac{1}{i + k} + \frac{1}{k + j} \ge \frac{1}{i+ j} \impliedby i^2 + ik + jk + ji + j^2 \ge k^2\] So it actually is untrue. Never mind, then!
