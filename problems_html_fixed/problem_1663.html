Problem 1663 - 2019 Autumn ToT Senior A-Level, P7 of 7 , N

Difficulty: 7

Some of the integers \(1,2,3,\dots,n\) have been colored red so that for each triplet of red numbers \(a,b,c\) (not necessarily distinct) if \(a(b-c)\) is a multiple of \(n\), then \(b=c\). Prove that there are no more than \(\varphi(n)\) red numbers.
