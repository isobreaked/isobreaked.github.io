Problem 501 - 2019 FKMO Day 1, P3 of 6, N

Difficulty: 7

For a positive integer \(x\), define a sequence \(a_0, a_1, a_2, \dots\) according to the following rules: \[a_0 = 1, a_1 = x+1 \text{ and } a_{n+2} = xa_{n+1} - a_n \text{ for all } n \geq 0.\] Prove that there exist infinitely many positive integers \(x\) such that this sequence does not contain a prime number.
