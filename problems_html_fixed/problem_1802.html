Problem 1802 - 2024 AMO, P8 of 8, N

Difficulty: 6

Let \(r = 0.d_0d_1d_2 \ldots\) be a real number (written in decimal form) where \(d_0, d_1, d_2, \ldots\) is an infinite sequence of digits. For each integer \(n \geq 0\), let \[e_n = 10^nd_n + 10^{n-1}d_{n-1} + \cdots + 10d_1 + d_0\] be the number formed by writing the digits \(d_n\), \(d_{n-1}\), \(\dots\), \(d_1\), \(d_0\) in order from left to right (leading zeros are permitted). Suppose that \(d_0 = 6\) and, for each integer \(n \geq 0\), the number \(e_n\) is equal to the number formed by the rightmost \(n + 1\) digits of \(e_n^2\). Prove that \(r\) is irrational.

Thinking

Clearly this is not true if the rational terminates (due to \(d_0 = 6\)). Thus we only need to consider the case where \(r\) has a periodic decimal expansion. Suppose the repetitive term is \[r = 0.(\overline{d_0d_1d_2 \dots d_{k-1}})\] \[e_{k-1}^2 \equiv e_{k-1} \pmod{10^k}\] \[e_{2k - 1}^2 \equiv e_{2k - 1} \pmod{10^{2k}} \implies e_{k-1}^2(10^k + 1)^2 \equiv e_{k-1}(10^k + 1) \pmod{10^{2k}} \implies e_{k-1}^2 (10^k + 1) \equiv e_{k-1} \pmod{10^{2k}}\] \[e_{3k - 1}^2 \equiv e_{3k - 1} \pmod{10^{3k}} \implies e_{k-1}^2(10^{2k} + 10^k + 1) \equiv e_{k-1} \pmod{10^{3k}}\] I am thinking of something involving \((10^{ik} - 1)/(10^k - 1)\) but that doesn't seem to be very helpful. I'll write it out for the sake of it: \[e_{k- 1}^2 \cdot \frac{10^{ik} - 1}{10^k - 1} \equiv e_{k-1} \pmod{10^{ik}} \implies -e_{k-1}^2 \equiv (10^k - 1) e_{k-1} \pmod{10^{ik}}\] Wait, this expression is completely independent of \(i\) and works for every \(i\). Taking \(i\) big, this means \(-e_{k-1}^2 = (10^k - 1)e_{k-1}\), which is impossible, unless \(e_{k-1} = 0\). But \(d_0 = 6\), so \(e_{k-1} \neq 0\) - thus \(r\) is irrational and we are done.

This isn't all rationals LOL. A general rational either terminates or is periodic after some random stuff at the start. Surely we have already obtained the main idea though. Suppose \(r\) looks like this: \[r = 0.(f_0f_1 \dots f_{s-1})(\overline{d_0d_1d_2 \dots d_{k-1}})\] \[e_{s-1}^2 \equiv e_{s-1} \pmod{10^s}\] \[e_{k-1}^2 \equiv e_{k-1} \pmod{10^{s+k}} \implies (10^s e'_{k-1} + e_{s-1})^2 \equiv (10^s e'_{k-1} + e_{s-1}) \pmod{10^{s+k}}\] \[e_{2k-1}^2 \equiv e_{2k-1} \pmod{10^{s+2k}} \implies (10^s e'_{2k-1} + e_{s-1})^2 \equiv (10^s e'_{2k-1} + e_{s-1}) \pmod{10^{s+2k}}\] \[\implies (10^s e'_{k-1} (10^k+1) + e_{s-1})^2 \equiv (10^s e'_{k-1}(10^k+1) + e_{s-1}) \pmod{10^{s+2k}}\] \[(10^s e'_{k-1} (\frac{10^{ik}-1}{10^k - 1}) + e_{s-1})^2 \equiv (10^s e'_{k-1}(\frac{10^{ik}-1}{10^k - 1}) + e_{s-1}) \pmod{10^{s+ik}}\] \[(10^s e'_{k-1} ({10^{ik}-1}) + (10^k - 1)e_{s-1})^2 \equiv (10^k - 1) (10^s e'_{k-1}(10^{ik}-1) + (10^k - 1) e_{s-1}) \pmod{10^{s+ik}}\] \[(e'_{k-1} (-10^s) + (10^k - 1)e_{s-1})^2 \equiv (10^k - 1) (e'_{k-1}(-10^s) + (10^k - 1) e_{s-1}) \pmod{10^{s+ik}}\] Great. We again have a congruence independent of \(i\), so varying \(i\) in the modulus, we get \[(e'_{k-1} (-10^s) + (10^k - 1)e_{s-1})^2 = (10^k - 1) (e'_{k-1}(-10^s) + (10^k - 1) e_{s-1}) \implies e'_{k-1} (-10^s) + (10^k - 1)e_{s-1}= 10^k - 1\] This means \(10^k - 1 \mid e'_{k-1}\), so we either reduce to the terminating case, or \(e'_{k-1} = 10^k - 1\), which further implies \[-10^s + e_{s-1} = 1\] simply absurd. We are done.

Idea

Famous last words: "that doesn't seem to be very helpful". Just write out everything, it is worth.

The solution looks long, but the main idea is complete before the realisation that not all rationals are terminating or periodic from the very start. Basically, actively looking for simpler problems may give insight (in this case, the main idea) into the full problem.
