Problem 909 - Folklore

Difficulty: 9

You wish to simulate an \(n\)-sided die using finitely many coin flips. For any \(p \in [0, 1]\) you can order a coin with bias \(p\) and flip that coin as many times as you wish. For each \(n\), what is the fewest number of coins needed? (For example, a \(4\)-sided die can be simulated with one coin by flipping a fair coin twice.)
