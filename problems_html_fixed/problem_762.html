Problem 762 - 2015 IMOSL, N7, N

Difficulty: 9

Let \(\mathbf{Z}_{>0}\) denote the set of positive integers. For any positive integer \(k\), a function \(f: \mathbf{Z}_{>0} \to \mathbf{Z}_{>0}\) is called \(k\)-good if \(\gcd(f(m) + n, f(n) + m) \le k\) for all \(m \neq n\). Find all \(k\) such that there exists a \(k\)-good function.
