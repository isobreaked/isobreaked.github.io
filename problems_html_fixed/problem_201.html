Problem 201 - 2010 China TST Day 3, P3 of 3 , N

Difficulty: 9

Let \(k > 1\) be an integer, set \(n = 2^k + 1.\) Prove that for any positive integers \(a_{1} < a_{2} < \dots < a_{n},\) the product \(\displaystyle\prod_{1 \leq i < j \leq n}^{} (a_{i} + a_{j})\) has at least \(k + 1\) distinct prime factors.
