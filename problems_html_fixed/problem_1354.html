Problem 1354 - 1986 IMO, P5 of 6

Difficulty: 6

Find all functions \(f\), defined on the non-negative real numbers and taking non-negative real values, such that:

  1. \(f(xf(y))f(y) = f(x+y)\) for all \(x,y\ge 0\)
  2. \(f(2) = 0\),
  3. \(f(x) \neq 0\) for \(0\le x < 2\)

Thinking

Let \(x = 0\), giving \(f(0)f(y) = f(y)\), and taking \(y < 2\), we must have \(f(0) = 1\). Now let \(y = 2\) and \(x\) be arbitrary, giving \(f(x + 2) = 0\) for all \(x \in \mathbf{R}_{\ge 0}\).

Let \(x = 1\), giving \(f(y) f(f(y)) = f(y + 1)\). If \(1 \le y \le 2\), then \(f(f(y)) = 0 \implies f(y) \ge 2\). If \(0 \le y < 1\), then \(f(y) < 2\).

Suppose \(y < 2\), that is, \(f(y) \neq 0\). Let \(x = 2/f(y)\), which gives \(f(y + {2}/{f(y)}) = 0\), so \(y + 2/f(y) \ge 2 \implies f(y) \le 2/(2-y)\). We know that \(f(1) \ge 2\), and from here \(f(1) \le 2\), so \(f(1) = 2\).

[absolute nonsense begins] So \(2f(2y) = f(y+1)\). \(x = y\) gives \(f(yf(y))f(y) = f(2y)\). \[f(y)f(f(y)) = f(y+1) = 2f(2y) = 2f(y)f(yf(y)) \implies f(f(y)) = 2f(yf(y))\]

This might be generalized symmetry, so I'll do that \[f(xf(y)) f(y) = f(x + y) = f(yf(x)) f(x)\] Wait bruh \(f(2f(y))f(y) = f(2+y) = 0\). So if \(f(y) \neq 0\), that is, \(y < 2\), then \(f(2f(y)) = 0\), which means \(2f(y) \ge 2\) or \(f(y) \ge 1\) for all \(y < 2\). \[1 \le f(y) \le 1+\frac{y}{2-y}\] [absolute nonsense ends]

Interestingly, we get \(f(xf(2-x)) = 0 = f((2-x)f(x))\) for all \(0 \le x < 2\), so \((2-x)f(x) \ge 2\), which means \(f(x) \ge 2/(2-x)\). Thus we have a unique solution that works \[f(x) = \begin{cases} & 2/(2-x) & \forall x < 2 \\ & 0 & \forall x \ge 2 \end{cases}\]
