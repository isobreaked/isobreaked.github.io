Problem 2370 - 1995 USAMO, P4 of 6, N

Difficulty: 6

Let \(q_0\), \(q_1\), \(q_2\), \(\dots\), be an infinite sequence of integers satisfying the following two conditions:

  • \(m-n\) divides \(q_m - q_n\) for \(m > n \ge 0\),
  • there is a polynomial \(P\) such that \(|q_n| < P(n)\) for all \(n\).
Prove that there is a polynomial \(Q\) for which \(q_n = Q(n)\).
