Problem 1783 - Unknown

Difficulty: 7

A word is a sequence, either finite or infinite, composed of the symbols \(0\) and \(1\), which are called letters. Given a word, a subword is a collection of consecutive letters.

An infinite collection of words is given: the first one has \(10\) letters, the second has \(100\) letters, the third has \(1000\), and so on. Prove that there exists an infinite word which doesn't contain any word from the collection as a subword.
