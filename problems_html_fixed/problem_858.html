Problem 858 - 2021 APMO, P2 of 5, N

Difficulty: 7

For a polynomial \(P\) and a positive integer \(n\), define \(P_{n}\) as the number of positive integer pairs \((a, b)\) such that \(a < b \leq n\) and \(|P(a)|-|P(b)|\) is divisible by \(n\). Determine all polynomial \(P\) with integer coefficients such that \(P_{n} \leq 2021\) for all positive integers \(n\).
