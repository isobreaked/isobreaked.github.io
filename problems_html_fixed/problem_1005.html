Problem 1005 - 2019 India TST, Day 4, P1 of 3

Difficulty: 7

Determine all non-constant monic polynomials \(f(x)\) with integer coefficients for which there exists a natural number \(M\) such that for all \(n \geq M\), \(f(n)\) divides \(f(2^n) - 2^{f(n)}\).

Thinking

I thought about some real divisibility things like somehow forcing powers of \(2\) but that seems very weird. Perhaps size gives something (note that \(f\) is positive for sufficiently large \(n\)): \[f(n) \le |f(2^n) - 2^{f(n)}|\] Actually this is trivial - because \(2^{n^k}\) easily outpowers \(2^{nk}\). That is, \(2^{f(n)}\) is clearly far bigger than \(f(2^n)\) unless \(k= 1\) and \(f\) is linear or something.

Let us go back to ranodm divisibility things \[f(2^n) \equiv 2^{f(n)} \pmod{f(n)}\] So we know what modulo \(f(2^n)\) is as a function of \(f(n)\). \[f(2^n) = (2^n)^k + a_{k-1}(2^n)^{k-1} + a_{k-2}(2^n)^{k-2} + \dots + a_2(2^n)^2 + a_1 (2^n) + a_0\] \[f(2^{n+1}) = (2^{n+1})^k + a_{k-1}(2^{n+1})^{k-1} + a_{k-2}(2^{n+1})^{k-2} + \dots + a_2(2^{n+1})^2 + a_1 (2^{n+1}) + a_0\] There is no apparent easy way to relate these quantities. However, note that \[f(2^n) = (2^k)^n + a_{k-1}(2^{k-1})^n + a_{k-2}(2^{k-2})^{n} + \dots + a_2(2^2)^n + a_1 (2)^n + a_0(1)^n \implies f(2^n) \equiv f(2^{n \pmod{\phi(f(n))}}) \pmod{f(n)}\] \[2^{f(n)} \equiv 2^{f(n) \pmod{\phi(f(n))}} \pmod{f(n)}\] Notably, this is only true for when \(f(n)\) is odd.

We would love to get a value such that \(f(n) = p\) - which unfortunately cannot be true for some polynomials. So we take some \(p \mid f(n)\) and get \[f(2^n) \equiv f(2^{n \pmod{p - 1}}), \;\;\;\;\;\;\;\; 2^{f(n)} \equiv 2^{f(n) \pmod{p - 1}}\] Now we just have some random prime and information modulo this random prime - we should probably get some other information modulo this prime - note that \(p \mid f(n + kp)\) since \(p \mid f(n)\), so we can get some more information. \[f(2^{n + kp}) \equiv f(2^{n + k \pmod{p -1}}), \;\;\;\;\;\;\;\; 2^{f(n + kp)} \equiv 2^{f(n+k) \pmod{p-1}}\] Wait but this is actually wild - we have obtained that \(f(2^{n+k \pmod{p-1}}) \equiv 2^{f(n+k) \pmod{p-1}} \pmod{p}\) for all \(k\) - but they are also congruent if we replace \(p\) with every prime \(q\) dividing \(f(n+k)\). Since the prime set of \(f\) is infinite, we can choose infinitely many sufficiently large primes while keeping \(n\) fixed, and obtain that \[f(2^n) = 2^{f(n)}\]

But now, as mentioned above, size tells us that \(f\) must be linear - suppose \(f(n) = n + c\), then \[2^{n} + c = 2^{n +c} \implies c = 0 \implies f(n) = n\] This function works.

Idea

In general, whenever you have some random exponential composite modulus congruence and find it slightly inconvenient or just useless to use \(\phi(n)\), think of a prime divisor \(p\) of the modulus.

Remember, when you have a congruence modulo some random prime, you should want to obtain other stuff modulo the same random prime - this 'cancels out' the randomness and makes it global generalizable information.
