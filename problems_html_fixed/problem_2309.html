Problem 2309 - 2022 Japan MO Finals, P4 of 5, N

Difficulty: 7

Find all positive integer pairs \((x, y)\) such that \(3^x - 8^y = 2xy + 1\).

Thinking

I suppose \((4, 2)\) is the only working pair. If \(x\) is even, then \[\nu_2(3^x - 1) = \nu_2(x) + 2 = \min(y, \nu_2(2xy)) \implies 2 \mid y\] If \(x\) is odd, then \[\nu_2(3^x- 1) = 1 = \min(y, \nu_2(2xy)) \implies 2 \nmid y\] Okay, let's look at \(\nu_3\) then. We have to take \(2 \nmid y\), and we get \[\nu_3(8^y + 1) = 2 + \nu_3(y) = \nu_3(3^x - 2xy)\] This is not possible. So \(2 \mid y\), \(2 \mid x\), \(3 \nmid x\), \(3 \nmid y\).

So letting \(x = 2a\), \(y = 2b\), we have \[(3^a - 8^b)(3^a + 8^b) = 8ab + 1\] We know that unless \(a = 2\) and \(b = 1\), \(3^a - 8^b \ge 5\). Since \(a > b\), we have \(3^a < 8a^2/5\), which just never holds. So the only working pair is \((a, b) = (2, 1)\), which means \((x, y) = (4, 2)\).

Idea

A direct size based approach does not work, a 'take \(p \mid x\)' or similar approach does not work, so the only option is \(\nu_3\) and forced difference of squares.
