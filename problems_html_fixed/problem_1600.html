Problem 1600 - 2023 Dutch TST day 1, P3 of 4, N

Difficulty: 6

Find all positive integers \(n\) such that there exist \(n\) distinct positive integers, none of which greater than \(n^2\), so that the sum \[\frac{1}{a_1} + \frac{1}{a_2} + \dots + \frac{1}{a_n} = 1.\]

Thinking

Thinking about primes. Suppose we have the triple \(2, a, b\) that works. Then \(2ab \mid 2a + 2b + ab \). So \(2 \mid a\) or \(2 \mid b\) must hold, suppose \(a = 2a'\). Then \(4a'b \mid 4a' + 2b + 2a'b\), which means \(2a'b \mid 2a' + b + a'b\). So we don't really get anything about \(2\)-divisibility apart from that at least one other number must be divisible by it.

Suppose the tuple \(2,a,b,c\) works. Then \(2abc \mid 2ab + 2bc + 2ca + abc\). Suppose \(a = 2a'\). Then \(2a'bc \mid 2a'b + bc + 2ca' + a'bc\).
