Problem 1258 - 2011 ToT Spring, Junior A-7

Difficulty: 7

In every cell of a square table is a number. The sum of the largest two numbers in each row is \(a\) and the sum of the largest two numbers in each column is \(b\). Prove that \(a=b\).

Idea

We want to somehow take some sort of global summation, but don't know what to sum, where to sum, etc. Look at the greatest two elements of every single row - they sum up to \(an\). The best case scenario would be that this set is exactly the same set as the greatest two elements of every single column - but that doesn't have to be true.

The neat thing about grids is that what holds for columns holds for rows - so theoretically, showing \(a \ge b\) would be enough by symmetry. But let's not let that lousy reason be our motivation. If we were able to find a smaller subset of \(k\) columns and \(k\) rows where the top 2 elements of the \(k\) columns form the same set as the top 2 elements of the \(k\) rows, we would be done.

The main idea here is that we don't need it to be the same set as the top 2 elements of the \(k\) rows, we only need it to be the same set as some two elements of the rows. This would fetch us the inequality \(bn \ge an\) and we can hope by symmetry to obtain a similar set for columns.

Such a set does exist, and is the same as the following problem from IMC 1999:
``Suppose that \(2n\) points of an \(n \times n\) grid are marked. Show that for some \(k > 1\) one can select \(2k\) distinct marked points, say \(a_1\), \(\dots\), \(a_{2k}\), such that \(a_{2iâˆ’1}\) and \(a_{2i}\) are in the same row, \(a_{2i}\) and \(a_{2i+1}\) are in the same column, \(\forall i\), indices taken mod \(2n\).''
