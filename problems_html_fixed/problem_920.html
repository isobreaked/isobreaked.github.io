Problem 920 - 2018 IMOSL, C1

Difficulty: 6

Let \(n\geqslant 3\) be an integer. Prove that there exists a set \(S\) of \(2n\) positive integers satisfying the following property: For every \(m=2,3,\dots,n\) the set \(S\) can be partitioned into two subsets with equal sums of elements, with one of subsets of cardinality \(m\).

Idea

You really want to induct so you try out various strategies, such as trying to replace a single \(x\) in the \(2k - 2\) set, which fail. The reason the mentioned strategy fails is because there is not a one-to-one correspondence - you don’t know which partition \(x\) is going to end up in exactly. That’s why you introduce the elements that you know are going to end up in opposite subsets of the partition.
